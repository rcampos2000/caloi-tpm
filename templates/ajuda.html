❓ Ajuda e Guia de Uso

📋 Como preencher o Formulário de Manutenção

1

Acesse o Formulário

Clique em Formulário no menu lateral ou no botão de acesso rápido na tela Início.

2

Informe o Equipamento

Digite ou escaneie o código do equipamento (ex: EQ001). O nome será preenchido automaticamente. Caso não haja código, preencha manualmente o nome.

3

Preencha os dados da ocorrência

Informe: técnico responsável, setor, horário de parada, horário de liberação, motivo e solução. O sistema calculará automaticamente o total de horas paradas.

4

Assine e finalize

O técnico e o solicitante devem assinar nos campos de assinatura digital. Clique em Registrar Manutenção para salvar. O Excel é atualizado automaticamente.

👥 Perfis de Usuário

🛡 Administrador Admin
Acesso completo: preenche formulários, visualiza dashboard, gerencia usuários e configurações do sistema (técnicos, setores, senhas).
🔧 Técnico Técnico
Acesso ao formulário de manutenção e dashboard. Não acessa as Configurações do sistema.

❓ Perguntas Frequentes

📄 O Excel é atualizado automaticamente?
Sim. A cada registro finalizado o sistema salva automaticamente no arquivo Manutencao_TPM.xlsx, localizado na pasta Manutencao_TPM no Desktop. Se o arquivo estiver aberto, o sistema tentará salvar por até 5 vezes.
🔑 Esqueci minha senha. O que faço?
Solicite ao Administrador do sistema que redefina sua senha no menu Configurações → Usuários → Editar.
➕ Como adicionar um novo técnico à lista do formulário?
Acesse Configurações → aba Técnicos, clique em "Adicionar Técnico", preencha o nome e clique em Salvar Alterações. A lista do formulário será atualizada imediatamente.
💾 Onde ficam os backups?
O sistema cria backups automáticos a cada 10 registros, salvos na pasta Manutencao_TPM/backups/. Os 10 backups mais recentes são mantidos.
📊 Como exportar os dados para Excel?
Clique em Exportar Excel no menu lateral ou acesse /download. Um backup é criado automaticamente antes do download.
⚡ O sistema não está abrindo. O que fazer?
Feche o servidor e abra novamente pelo arquivo INICIAR.vbs. Certifique-se que o Python está instalado e o PowerShell tem permissão de execução.
